# GPS for GBS 🗺️📍

## Overview
**GPS for GBS** is a dedicated Android navigation application designed to make finding your way around the Goa Business School (GBS) campus simple and efficient. Whether you are a first-year MCA student trying to locate a specific lab, a visitor searching for a faculty cabin, or just exploring the university grounds, this app provides clear, on-the-go guidance to get you exactly where you need to be.

## Features
* **Interactive Campus Map:** A custom, digital layout of the GBS building, departments, and surrounding university areas.
* **Point-to-Point Navigation:** Get direct routes from your current location to classrooms, computer labs, the library, or administrative offices.
* **Quick Search:** Easily look up specific room numbers, faculty names, or department wings.
* **Accessible UI:** Clean, intuitive interface designed for quick use while walking across campus.

## Tech Stack
* **Frontend:** XML for responsive and visually appealing user interfaces.
* **Backend/Logic:** Java
* **Development Environment:** Android Studio

## Prerequisites
To run this project locally, you will need:
* **Android Studio** (latest stable version recommended)
* **Android SDK** (API level 24 or higher recommended)
* A physical Android device or an Android Virtual Device (AVD) emulator for testing.
